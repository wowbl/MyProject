# MyProject

To install the library you need the pip program:
Debian/Ubuntu
```sh
$ sudo apt-get install python-pip
```
Once you have the pip program issue the following commands:
```sh
$cd flasky
$sudo pip install -r requirements.txt
```
Setup Mosquitto
```sh
$cd flasky/mosquitto-1.4
$make all
$sudo make install
$sudo ldconfig
```

Star Middleware
```sh
$ run.py
```
